# Альбомы исполнителя "Дайте танк (!)"

Здесь собрана большая часть альбомов исполнителя "Дайте танк (!)".

Каждый файл `.mp3` содержит:
*   Название композиции
*   Исполнителя
*   Название альбома

Каждый альбом имеет кастомную иконку.

### Инструкции по использованию:

1.  **Скачайте архив:**
    [Скачать здесь](https://github.com/Sabackeny/daite-tank./raw/refs/heads/master/%D0%94%D0%B0%D0%B9%D1%82%D0%B5%20%D1%82%D0%B0%D0%BD%D0%BA%20(!)%2011.zip?download=)

2.  **Распакуйте архив** в любую удобную папку на вашем компьютере.

3.  Внутри папки вы увидите файл **`apply_icons.bat`**. Запустите его **двойным кликом**.

4.  Возможно, потребуется перезагрузить Проводник Windows или компьютер, чтобы иконки появились.

Приятного прослушивания!
