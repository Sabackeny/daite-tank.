# Альбомы исполнителя "Дайте танк (!)"

Здесь собрана большая часть альбомов исполнителя "Дайте танк (!)".

Каждый файл `.mp3` содержит:
*   Название композиции
*   Исполнителя
*   Название альбома

Каждый альбом имеет кастомную иконку.

### Инструкции по использованию:

1.  **Скачайте архив:**
2.  нажмите на releases и выберете первый
, либо скачайте по прямой [ссылке](https://github.com/Sabackeny/daite-tank./releases/download/v2/default.zip)

3.  **Распакуйте архив** в любую удобную папку на вашем компьютере.

4.  Внутри папки вы увидите файл **`apply_icons.bat`**. Запустите его **двойным кликом**.

5.  Возможно, потребуется перезагрузить Проводник Windows или компьютер, чтобы иконки появились.

Приятного прослушивания!
